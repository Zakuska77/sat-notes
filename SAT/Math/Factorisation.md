# Factoring Rules( Snippets )

1. $a^2 - b^2$ => $(a+b)(a-b)$
2. $a^2+2ab+b^2$ => $(a+b)^2$
3. $a^2-2ab+b^2$ => $(a-b)^2$

---

4. $ax+bx+c$ =>  $2x^2+7x+3$ 

> [!hint] 
> $n_{1} = a*c$
> $n_{2} = a+c = b$
> -> $n_{1} = 6*1$
> -> $n_{2} = 6+1 = 7$

  $2x^2+7x+3 = 2x^2 +6x+x+3 =$
  $2x(x+3)+1(x+3)$
