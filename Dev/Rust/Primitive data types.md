# Primitive data types

### Int

u -> unsigned => can't be negative
i -> signed => can be negative

Ex:
```
let x:i23 = -42;
let y:u32 = 42;
```

---
### Float
float =  fractions
`let z:f32 = 42.5;`

---
### Char 
single character like => "a", "1", etc.
`let ch:char = 'a';`

---
### Boolean
value that can be **True/False**

`let is_snowing:bool =  true;`
