### Important Keyword

**var** → normal changing variables
**final** → no change after (at the runtime level)
**const** → no change after (at the compile time level)
**void** → fucntion with no return

### Types
**String** → characters 
**Int** → integers
**boul** → true/false
**double** → float

### Example 

```
const int age = 22
var String name = "Alex"
```
