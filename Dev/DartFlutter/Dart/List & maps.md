### List
A list is an ordered collection of item
`List<String> fruits = ['Apple', 'Banana', 'Orange'];`

### Maps
A map in Dart is an unordered collection of key-value pairs.

```
Map<String, int> ages = { '
'Alice': 30, 
'Bob': 25, 
'Charlie': 35, 
};

print('Alice\'s age: ${ages['Alice']}'); // alice here is the key```
